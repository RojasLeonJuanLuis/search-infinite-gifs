# search-infinite-gifs
Busca tus gifs favoritos y haz scroll para tener una paginación infinita

## Install
```
$ git clone https://github.com/RojasLeonJuanLuis/search-infinite-gifs.git
$ npm install
$ npm start
```
## Production
```
$ npm run build
$ (Use your favorite tool for production)
```
